Random (Erdős-Rényi) Graphs¶

Create an G{n,m} random graph with n nodes and m edges and report some properties. This graph is sometimes called the Erdős-Rényi graph but is different from G{n,p} or binomial_graph which is also sometimes called the Erdős-Rényi graph.

import networkx as nx

%matplotlib inline

G = nx.gnm_random_graph(10, 20)   # 10 nodes and 20 edges

for v in G.nodes():   # Some properties of the graph
    print(v, G.degree(v), nx.clustering(G,v))
